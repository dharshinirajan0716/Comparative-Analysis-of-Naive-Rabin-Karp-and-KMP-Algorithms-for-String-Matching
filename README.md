# Comparative-Analysis-of-Naive-Rabin-Karp-and-KMP-Algorithms-for-String-Matching
A Python-based Streamlit application that compares Naive, Rabin-Karp, and Knuth-Morris-Pratt (KMP) string matching algorithms by analyzing match positions, character comparisons, and performance on different input patterns.

## Overview
This project compares three popular string matching algorithms:
Naive String Matching,
Knuth-Morris-Pratt (KMP),
Rabin-Karp.

The application allows users to enter a text and a pattern, then compares the algorithms based on:
Match positions,
Number of character comparisons,
Performance comparison.

## Technologies Used
Python,
Streamlit.

## Algorithms
Naive Algorithm,
KMP Algorithm,
Rabin-Karp Algorithm.

## How to Run
bash
pip install -r requirements.txt
streamlit run app.py
